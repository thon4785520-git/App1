# ระบบสารสนเทศยืมคืนอุปกรณ์ศาสนพิธี

เว็บแอป PHP 8 + MySQL + Bootstrap 4 สำหรับจัดการการยืม/คืนอุปกรณ์งานศาสนพิธี รองรับ 2 บทบาท: `admin` และ `user`

## โมดูลหลัก
- จัดการข้อมูลผู้ใช้งาน
- จัดการข้อมูลประเภทอุปกรณ์
- จัดการข้อมูลอุปกรณ์
- จัดการการยืมคืน
- ระบบรายงาน

## เทคโนโลยี
- HTML5
- Bootstrap 4.6
- PHP 8 (PDO)
- MySQL 8+

## วิธีติดตั้ง
1. สร้างฐานข้อมูลโดย import ไฟล์ `database/schema.sql`
2. แก้ค่าการเชื่อมต่อใน `config/database.php`
3. รันผ่าน Apache/Nginx + PHP 8 (DocumentRoot ชี้มาที่โฟลเดอร์นี้)

## บัญชีทดสอบ
- ผู้ดูแลระบบ: `admin` / `password`
- ผู้ใช้ทั่วไป: `user1` / `password`

## โครงสร้างหลัก
- `admin/` หน้าจัดการสำหรับผู้ดูแลระบบ
- `user/` หน้าการใช้งานของผู้ยืม
- `includes/` ส่วน header/footer/auth/menu
- `database/schema.sql` โครงสร้างฐานข้อมูลและข้อมูลเริ่มต้น
